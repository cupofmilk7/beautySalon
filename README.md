## Описание

**Beauty Salon** — это многомодульное приложение для автоматизации работы салона красоты. Система позволяет:

- Управлять клиентской базой и профилями пользователей.
- Создавать и отслеживать события (записи, напоминания, акции).
- Отправлять персонализированные SMS-уведомления клиентам.
- Администрировать систему через удобный веб-интерфейс.

Проект построен по принципам микросервисной архитектуры, что обеспечивает масштабируемость, гибкость и независимость
компонентов.

---

## Архитектура

Система состоит из следующих микросервисов:

| Сервис            | Описание                              |
|-------------------|---------------------------------------|
| **user-service**  | Управление пользователями и клиентами |
| **event-service** | Управление событиями и отправка SMS   |
| **admin-service** | Административный интерфейс            |

Коммуникация между сервисами осуществляется через **Feign Client** (синхронное взаимодействие) и **REST API**.

---

## Сервисы

### 1. User Service

- CRUD операции для пользователей.
- Фильтрация пользователей по дате рождения (год).
- Валидация данных (email, телефон).

### 2. Event Service

- Управление событиями (создание, обновление, удаление, просмотр).
- Получение активных событий (по дате).
- Интеграция с **SMS Aero API** для отправки сообщений.
- Фильтрация пользователей по датам для массовых рассылок.

### 3. Admin Service

- Административная панель на основе **SnapAdmin**.
- Управление всеми сущностями системы (пользователи, события, фильтры).

## Функциональные возможности

### Управление событиями

- Создание события с указанием:
    - Описания.
    - Даты начала и окончания.
    - Фильтра для выборки пользователей.
    - Даты отправки сообщения.
- Получение списка событий по дате.
- Получение активных событий.
- Обновление и удаление событий.

### SMS-рассылка

- Автоматическая отправка сообщений клиентам при создании события.
- Использование **SMS Aero API** для отправки.
- Поддержка тестового режима отправки.
- Настраиваемая подпись отправителя.

### Администрирование

- Визуальное управление данными через **SnapAdmin**.
- Фильтрация и поиск по сущностям.
- Просмотр истории изменений.

## Технологический стек

| Компонент       | Технология                  |
|-----------------|-----------------------------|
| Язык            | Java 21                     |
| Фреймворк       | Spring Boot 3.3.5           |
| ORM             | Hibernate / Spring Data JPA |
| База данных     | PostgreSQL                  |
| Микросервисы    | Spring Cloud (Feign Client) |
| Админ-панель    | SnapAdmin                   |
| SMS-шлюз        | SMS Aero API                |
| Сборка          | Maven                       |
| Контейнеризация | Docker                      |

## Требования

- **JDK 21** или выше.
- **PostgreSQL** 14+.
- **Maven** 3.8+.
- **Переменные окружения** для SMS-шлюза.
