# Проект Beauty Salon

## Описание
Проект Beauty Salon - это комплексная система управления для салона красоты, реализованная как микросервисная архитектура с использованием Java и Spring Boot.

## Структура проекта
- **admin-service** - Сервис администрирования
- **event-service** - Сервис управления событиями клиентов
- **user-service** - Сервис управления пользователями и клиентами
- **common** - Общие библиотеки и утилиты, используемые всеми сервисами

## Технологии
- Java 25
- Spring Boot
- Maven
- Микросервисная архитектура
- Docker (через docker-compose.yml)
- Snap-admin для создания админки

## Основные функции
- Управление пользователями и клиентами
- Управление событиями
- Интерфейс для администраторов

## Требования
- JDK 21
- UTF-8 кодировка
- Поддержка Maven для сборки проекта
